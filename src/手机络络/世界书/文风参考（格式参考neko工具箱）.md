writing_style_guide:  
title: "情感类轻小说文风指南"  
sections:  

- title: "语言与表达"  
  description: "细腻生动：用诗意和俏皮的语言传递情感。"  
  guidance: "与其直接表达‘我喜欢你’，不如用意象描绘，比如‘你的笑容像春日的暖风，轻轻拂过我的心田’。"  
  example: "“我喜欢你。”她的声音轻得像一片羽毛，飘进了我的耳朵，却重重地落在了我的心上。"（改编自《文学少女》）  

- title: "动作与姿态"  
  description: "自然流畅：动作应反映角色的情绪与个性。"  
  guidance: "通过细微的动作展现内心，比如紧张时的手指轻扣，或开心时的轻快步伐。"  
  example: "她的手指在裙摆上轻轻揉搓，像是在掩饰什么，又像是在等待什么。"（灵感来源于《春物》中的细腻动作描写）  

- title: "场景与氛围"  
  description: "梦幻唯美：用迷人的场景映射角色的情感。"  
  guidance: "描绘充满色彩与声音的场景，让读者沉浸其中。"  
  example: "樱花如雨般洒落，风轻轻拂过，带着一丝甜蜜的花香，她站在树下，仿佛与这片粉色的世界融为一体。"（灵感来源于《秒速五厘米》的场景描写）  

- title: "内心独白"  
  description: "俏皮深刻：用富有想象力的语言展现角色的内心世界。"  
  guidance: "通过奇妙的比喻或诗意的语言表达角色的思绪。"  
  example: "‘如果心跳也能被听见的话，那现在的我一定像一只被困在玻璃瓶里的小鸟，扑腾得毫无章法。’"（灵感来源于《四月是你的谎言》的内心描写）  

- title: "外貌与风格"  
  description: "个性鲜明：通过服饰与外貌展现角色的独特性。"  
  guidance: "用鲜明的色彩、质感或配饰突出角色的个性。"  
  example: "她穿着一件白色的连衣裙，裙摆上点缀着细碎的蓝色花纹，像是夏日天空中漂浮的云朵。"（灵感来源于《文学少女》的角色描写）  

- title: "对话与互动"  
  description: "节奏鲜明：让对话充满趣味与情感。"  
  guidance: "通过俏皮的对话或深情的交流展现角色间的关系。"  
  example: "‘你为什么总是这么笨？’她轻轻笑着，眼中却满是宠溺。‘因为我知道你会帮我啊。’他挠了挠头，笑得像个孩子。"（灵感来源于《龙与虎》的对话风格）  

- title: "情感发展"  
  description: "自然递进：让情感随着时间和互动逐渐加深。"  
  guidance: "从细微的互动开始，逐渐发展为深厚的情感连接。"  
  example: "他们的手指不经意间碰触了一下，像是电流划过，短暂却让人久久无法忘怀。"（灵感来源于《未闻花名》的情感递进）  

- title: "苦甜交织"  
  description: "温柔忧伤：在角色的情感中融入一丝渴望与遗憾。"  
  guidance: "通过细腻的瞬间展现角色的内心冲突与情感深度。"  
  example: "‘如果当时我能再勇敢一点，现在的我们会不会不一样？’他看着远处的夕阳，眼中闪过一丝不易察觉的遗憾。"（灵感来源于《你的名字》的情感描写）  

- title: "经典语句"  
  description: "引发共鸣：融入能唤起情感的经典短句。"  
  guidance: "用简洁而深刻的语言触动读者的心弦。"  
  example: "‘世界上有一种无法挽回的东西，那就是过去的时光。’"（出自村上春树《挪威的森林》）  

- title: "情感层次"  
  description: "丰富细腻：通过动作、对话与内心描写展现复杂情感。"  
  guidance: "结合多种表现手法，创造情感的深度与层次。"  
  example: "‘我想要靠近你，却又害怕靠得太近会让你发现我的不完美。’她低下头，手指紧紧攥着衣角。"（灵感来源于《白夜行》的情感层次描写）
