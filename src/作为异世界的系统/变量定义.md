# 变量定义一览

## 世界

- `世界.当前时间`：初始化为“待初始化”，记录当前叙事时间。
- `世界.当前地点`：初始化为“待初始化”，标识主角所处地点。

## 主角

- `主角.姓名` / `性别` / `年龄`：基础身份字段，默认“待初始化”。
- `主角.外貌.描述`、`主角.外貌.自我评价`：用于展示形象与主观感受。
- `主角.能力面板.[力量|敏捷|体质|感知|意志|魅力].数值`：六维属性数值。
- `主角.能力面板.[力量|敏捷|体质|感知|意志|魅力].自我评价`：主角对对应属性的自评。
- `主角.物品栏`：数组，元素结构为 `[名称, 描述, 主角评价]`。
- `主角.装备栏.主手` / `副手` / `防具` / `饰品`：各装备位当前装备名称。

## 系统状态

- `系统状态.系统模式`：描述当前系统运行模式。
- `系统状态.可用积分`：主角可支配的系统积分。
- `系统状态.当前提示`：系统即时提示或指引。
- `系统状态.对主角可见形象`：系统对主角展示的形象。
- `系统状态.玩家ID`：绑定玩家标识，默认 `{{user}}`。

## 任务列表

- `任务列表`：可扩展列表；默认模板包含 `任务名`、`任务类型`（默认“主线”）、`任务描述`。
